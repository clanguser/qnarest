Steps:
1. Clone the repo
2. Install the packages
'''
  npm init -y
  npm install express mongoose nodemon jsonwebtoken bcrypt
'''

3. Run the server
'''
  npx nodemon server.js
'''
4. Use route.rest to test the API endpoints
